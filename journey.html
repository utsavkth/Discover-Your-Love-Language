The Development Journey

Many people often ask me if this website is part of a university project or what motivated me to create it. Before launching this website, I developed a Google Form to calculate love language results manually. As I marked each love language for every question on my iPad, I thought, "What if I could create a program or web app to automate this process?" This idea sparked the creation of my website.

The journey involved multiple iterations and designs, each one refined through feedback from friends and colleagues. I am particularly grateful to Alvina, for whom I initially intended the entire website; her insights were invaluable in shaping the project. Special thanks also go to Aaditya, whose constructive feedback prompted numerous changes, particularly in the user interface and question presentation. I cannot count the number of prototypes stored on my PC!

This project started as a simple design using basic HTML and CSS. After creating the Google Form, I spent about 5-6 hours developing the initial version. With a full-time job and working double shifts due to circumstances, I dedicated around 10 days to get to the current version, enhancing it with JavaScript and CSS along the way. Emphasizing Privacy and Transparency

Creating a website may seem daunting, with many people insisting that you need to buy a domain or set up a database. However, I proved that it can be done using GitHub and Formspree for feedback collection. I plan to continue updating this website and appreciate everyone's support and encouragement throughout this journey.

One significant realization during my work with the Google Form was the importance of user privacy. I felt uncomfortable receiving personal love language results, which further motivated me to develop this website. Now, users can see their love language results as they progress through the quiz, and I have included a privacy note on the quiz page to address concerns about data collection. Some individuals even declined to participate in the Google Form quiz due to privacy worries, so I wanted to create a more transparent and respectful experience.
